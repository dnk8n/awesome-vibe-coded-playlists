# Dance Mania 1994–1996: A Mini Label Retrospective

Five essential releases from the Chicago label's ghetto-house golden run, max one per artist. Source of truth: [discogs.com](https://www.discogs.com/label/314-Dance-Mania) (label ID 314). Artist and track names are reproduced exactly as credited on each Discogs release page (the trailing `*` is Discogs's own marker for an artist-name variation). Discogs lists year-only release dates for all five records, so rows are sorted by year, then catalog number (which tracks the label's release sequence). Every YouTube link below was taken from the videos embedded on that release's Discogs page and validated via the YouTube Data API on 2026-07-17. All five release pages had videos, so no entry needed its video skipped.

## Notes

- **One artist per row** enforced: DJ Deeon, DJ Funk, Parris Mitchell, Paul Johnson, and Wax Master (with DJ Rated-X) each appear once, despite all having multiple 1994–1996 Dance Mania releases.
- **Release dates**: Discogs provides only the year for all five original pressings; no month/day data exists on the release pages.
- **Row 3 track credit**: the Discogs release page credits the release to Parris Mitchell, with "Featuring – Reggie Hall" listed under track A2 "All Night Long".
- **Row 5 track credit**: the release is credited to "Wax Master* & DJ Rated-X"; track B1 "Foot Work" is individually credited to Wax Master* on the release page.
- Validated YouTube IDs, in table order, are in `playlist_items.json`. No YouTube playlist was created (no OAuth available).
