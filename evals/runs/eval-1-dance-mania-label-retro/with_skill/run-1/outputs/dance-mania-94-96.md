# Dance Mania 1994–1996 — five essential releases

**Brief:** a mini label retrospective — 5 essential releases from Dance Mania (Chicago), 1994–1996, max one release per artist. Source of truth: discogs.com — every row below was verified against the Discogs API (release fetch: credited artist, tracklist, label/cat#, page videos). Artist and Track are copied exactly as credited on the linked release page.

**Scope note:** this window is the label's ghetto house golden era, so the five slots go to the five artists who defined it — DJ Deeon, Paul Johnson, DJ Funk, Parris Mitchell, DJ Slugo. One track is picked per release as the representative cut.

**Sorting:** Discogs carries year-only dates for these pressings, so rows sort by year, then by Dance Mania catalog number within a year.

**YouTube links:** all five come from the `videos[]` embedded on each release's own Discogs page (community-matched), and every id was validated via the YouTube API (5/5 public). No release page lacked a video, so nothing had to be skipped.
